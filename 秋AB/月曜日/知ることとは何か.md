### ハイデガーの思想
* 「存在と時間」
    * 理解のもっている循環的構造を指摘
    * 「予」構造
        * 私たちを物ごとを「常に既に」知っている
        * 私たちが何かを認識するときに、私たちは客観的に認識するのではなく、予め得ている理解に基づいて認識していく
        * 予めストックされている知識の中から、個々のものをどういうふうに理解するかということが出てきて、逆に個々の理解から全体とはどういうものなのかということが出てくる
        * 「予・構造」が上手くいかなくなったときに解釈が発生する
    * 全体と部分
        * 部分の理解があることで全体の理解が生まれる
            * 小説は、前から読んでいく(部分の理解)ことによって小説の真髄を理解(全体を理解)する
        * 逆に、全体を理解して部分を理解することもある
            * 小説を最後まで読みきった後(全体の理解)に、ある部分に出てきた伏線の意味を理解(部分の理解)をするような話
### さまざまな思想
* ハンソン
    * 「観察の理論負荷性」

* クーン

### 啓蒙主義は本当に「先入観」を否定したのか
* ボルテール『哲学辞典』 (1764年)
    * 先入観とは判断を伴わぬ意見である。
    * 普遍的な必要な、美徳そのものの先入観がある。したがって、極めて有益な先入観というものが存在する。 それは理性が働くときに権威として認識されるものである
* 啓蒙主義の時点で、既に先入観に関する議論は盛んであり、「有益な先入観」という考え方も極めて一般的なものであった
* マイアー「先入観に対する先入観」
    * 先入観は全て悪だと思い込むこともまた先入観なのである
* ヘルダー『人間性形成のための歴史哲学異説』(1774年)
    * 様々な世界史上のできごとが私たちの人間性の表れなのではないか
    * 人類の子供時代としてのヘブライ史
        * <-> 啓蒙思想「過去の人類史は悪であり、現代に向かうにつれて改善されているものだ」
        * 人間の子供は理性によっては何も認知せず、先入観に頼った認識をするが、それはそれで子供時代には良いのではないか、という主張
        * 「先入観」: その時代にとっては善いもの。人類を幸福にする。
            * 先入観がその時代の人々の中心にあるものであるから

### 知ることとは何か
* 私たちは何事かについて知っているのは間違いないが、知ることは何かと問われると困惑する
* 伝統的教育を通じて知ることはできるが、それは本当に知っていることになるのだろうか
    * 一種の「迷信」に分類されるのではないか
* 「知る」
    * 「知識」 <-> 「臆見」
    * 「真理」 <-> 「意見」
    * 「判断」 <-> 「先入観」(偏見、予断)
 * 啓蒙思想における知の一般的特徴
    * それがある種の体系性を持っている
        * 「体系性(編集知)」 <-> 「断片性」
        * 神に与えられる知(断片的な知)ではなく、人類が得た知(体系的な知)を重要視する
    * 広く一般的に周知されるものである
        * 「公開性」 <-> 「私秘性」
        * 「知は力なり」という名言にもあるように、知識は権力となりうるため、知識は特殊な集団内で共有すべきものという伝統が存在した
        * 好奇心の肯定
    * 知は実践的なものである
        * 「実践知」 <-> 「理論知」

## 2017-11-20 知ることと信じること
### 学問とは
* 学問 = Science
    * science の中に study やら research やらが含まれる
    * science はラテン語派生
        * scientia: 12世紀に登場した誤
            * 世界初の大学ができたあたりの時期
    * scio = 知る + tia = 抽象化語尾 => scientia
    * 日本語に直訳すると「科学」
        * 「科学」には「知る」という概念が含まれていない
    * 「知る」とは何を対象としているのだろうか？
        * 「事実」である

### 事実とは何か
* 「知る」の対象が「事実」である限り、その「知る」という営みは "scientia" である
* アイザック・ニュートン「我は仮説を造らず」
    * 仮説を立て続けるのではなく、事実を淡々と収集し続ける
* data と fact
    * 研究者は fact という単語を絶対に用いない
        * 「リンゴが落ちた」という fact ではなく、「リンゴが落ちた」という data を用いる
    * fact
        * ラテン語 factum(過去分詞) => facere(原型)
            * 「作り出す」という意
                * factory, facture, fact などは ラテン語 facere からの派生
    * data
        * ラテン語 datum(過去分詞) => dare(原型)
            * 「与えられたもの」という意 
                * fact: 観察者に対して自然から与えられたもの(主観的ではない)
                    * 誰がやっても同じ手法を取れば必ず同じデータが「自然から与えられる」という事実の下、現代研究の基礎として君臨
        * data を取り集めて僕たちは何をしたいのか？ 
            * 法則を発見する
                * 法則を発見することこそ scientia である
        * data = 単称命題
* ケプラーの第一法則
    * **すべての**惑星は、太陽を1つの焦点とする楕円軌道上を動く
    * 当時知られていたのは 水・金・地・火・木・土
    * ケプラーは火星に興味を抱く
    * 望遠鏡とほぼ同時に顕微鏡が発明される
        * 望遠鏡が発展したのち、顕微鏡が発展
            * 赤血球、アメーバ、精子の発見などに欧州人は狂喜した
    * 当時、惑星は円軌道を描いていると思われていた
        * 円は「きれいな」形であり、神が作った天体は完全な円を描いているはずだと思われていたため
    * ケプラーは火星のみを観察して全ての惑星は楕円軌道を動くはずだと断定
        * この「帰納的な」断定手法が scientist が日常的に行う行為
* 帰納と演繹 / 普遍法則と予言可能性
    * 法則というのは予言可能性を持っている
        * 明日東京で x が起これば、 今日つくばで x が起こって y になったのと同様に、東京でも y となるはずだ
    * ケプラーの第一法則は「すべての」と言い切りつつ、実際には惑星たる6つの天体のみを対象にしているので、普遍法則とは言えないのではないか
    * 現代科学者は、何かの実験を有限回繰り返して、論文に全てのデータを載せる代わりに「省略」として「法則」を載せる
        * この考え方は「思考の経済」と呼ばれる(マッハ)
            * 思考の仕方はできるだけシンプルな方が良い、思考のアウトプットもできるだけシンプルな方が良いと無意識にされる
        * ここで言う普遍法則は「裏切られる」リスクを常にはらんでいる
            * 300回実験した中では法則が成立していたとしても、301回目に成り立たない可能性もある
* 普遍法則に伴う2つの altematives
    * 有限個の集合の和集合のことを「法則」というのか(ケプラーの第一法則のような)
    * 確証データに基づく無限個の性質の省略を「法則」というのか(現代科学者の予言可能性をはらんだ法則のような)
* 確証と反証の非対称性
    * 確証データは300個あっても同じデータであるから、衝撃力は弱い
    * 反証データは1つ存在するだけで全ての法則を打ち壊す破壊力を持つ
